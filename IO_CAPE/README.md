# Simple I/O cape.

Two 2x5 shrouded headers:

## Header J1: 

- Pin 1: GPIO 115 (yellow)
- Pin 2: GPIO 49  (green)
- Pin 3: GPIO 112 (grey)
- Pin 4: GPIO 117 (white)
- Pin 5: Ground (black)
- Pin 6: +3.3V  (pink)
- Pin 7: GPIO 47 (orange)
- Pin 8: GPIO 45 (ochre)
- Pin 9: GPIO 46 (cyan)
- Pin 10: GPIO 44 (brown)

## Header J2:

- Pin 1: GPIO 50 (EHRPWM1A) (yellow)
- Pin 2: GPIO 60 (green)
- Pin 3: GPIO 51 (EHRPWM1B) (grey)
- Pin 4: GPIO 48 (white)
- Pin 5: Ground (black)
- Pin 6: +3.3V  (pink)
- Pin 7: GPIO 69 (orange)
- Pin 8: GPIO 66 (ochre)
- Pin 9: GPIO 68 (cyan)
- Pin 10: GPIO 67 (brown)

# Simple I/O cape.

Two 2x5 shrouded headers:

## Header J1: 

- Pin 1: GPIO 115
- Pin 2: GPIO 49
- Pin 3: GPIO 112
- Pin 4: GPIO 117
- Pin 5: Ground 
- Pin 6: +3.3V
- Pin 7: GPIO 47
- Pin 8: GPIO 45
- Pin 9: GPIO 46
- Pin 10: GPIO 44

## Header J2:

- Pin 1: GPIO 50 (EHRPWM1A)
- Pin 2: GPIO 60
- Pin 3: GPIO 51 (EHRPWM1B)
- Pin 4: GPIO 48
- Pin 5: Ground 
- Pin 6: +3.3V
- Pin 7: GPIO 69
- Pin 8: GPIO 66
- Pin 9: GPIO 68
- Pin 10: GPIO 67
